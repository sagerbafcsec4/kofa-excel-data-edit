# サッカー記録 整形ツール — 使い方

フォルダ か Excelファイル を選んで「実行」を押すだけで、これまでの整形作業
（列幅・フォント・縮小表示・不要列/行/シートの削除・印刷範囲）を自動で行います。

- **元のファイルは変更しません。** 整形済みのコピーを、選んだフォルダの中の
  `_processed` フォルダに保存します。処理内容は `_logs` に記録されます。
- シートは名前ではなく**中身の構造**で自動判別します（シート名を変えても動作）。

## 配布する人（あなた）が最初に一度だけやること

1. このフォルダにある **`EXEを作成.bat`** を**ダブルクリック**します。
   - Python が入っていなければ自動で入れようとします（画面の指示に従ってください）。
   - 数分待つと、同じフォルダに **`Excelデータ加工.exe`** ができます。
2. この **`Excelデータ加工.exe`** を、使ってもらいたい人に渡すだけ。

> `EXEを作成.bat` と `soccer_format_app.pyw`、`format_records.py` は同じフォルダに
> 置いたまま実行してください（.exe を作るのに必要です）。

## 使う人（配布された人）がやること

1. もらった **`Excelデータ加工.exe`** を**ダブルクリック**。
   （Python など、何もインストールする必要はありません）
2. 「📁 フォルダを選ぶ」または「📄 ファイルを選ぶ」で対象を指定。
3. 必要なら「経歴シートを削除する」のチェックを調整。
4. **「▶ 実行」** を押す。終わると `_processed` フォルダが自動で開きます。

## 自分のPCですぐ試したいとき（任意）

Python が入っているなら、`アプリを起動.bat` をダブルクリックすると
.exe を作らなくてもアプリ画面が開きます。

## ファイルの役割

| ファイル | 役割 |
|---|---|
| `Excelデータ加工.exe` | 配布用アプリ（`EXEを作成.bat` で生成）。これだけ渡せばOK |
| `EXEを作成.bat` | .exe を作る（最初に一度だけ・ダブルクリック） |
| `アプリを起動.bat` | Python があればアプリを直接起動 |
| `soccer_format_app.pyw` | アプリの画面（GUI）本体 |
| `format_records.py` | 整形ロジック本体 |
| `build_exe.py` | EXEを組み立てる処理（batから自動で呼ばれる） |
| `build_log.txt` | ビルドの記録（うまくいかない時はこれを見せてください） |


## うまく作成できないとき

- `EXEを作成.bat` を実行しても黒い画面が一瞬で消える/exeができない場合、
  同じフォルダにできる **`build_log.txt`** を開いて内容を確認してください。
- 「Python not found」と出た場合は、画面の指示どおり一度 Python を入れてから、
  **もう一度** `EXEを作成.bat` をダブルクリックしてください。
- `EXEを作成.bat`・`build_exe.py`・`soccer_format_app.pyw`・`format_records.py` は
  必ず同じフォルダに揃えて置いてください。
